<task type="auto">
  <name>Task 1: Register config path and create historical_profiles.py module</name>
  <files>src/config.py, src/historical_profiles.py</files>
  <action>
  1. In src/config.py, add to SILVER_PLAYER_S3_KEYS:
     "historical_profiles": "players/historical/combine_draft_profiles_{ts}.parquet"
     Note: no season/week partition — this is a static dimension table.

  2. Create src/historical_profiles.py with these pure functions (NO I/O, NO file reads/writes):

  a) parse_height_to_inches(ht_str: str) -> Optional[float]
     - Convert "5-11" to 71.0 inches. Return None on NaN or parse failure.
     - Split on "-", int(feet)*12 + int(inches). Wrap in try/except.

  b) compute_speed_score(wt: pd.Series, forty: pd.Series) -> pd.Series
     - Bill Barnwell formula: (weight * 200) / (forty ** 4)
     - NaN propagates naturally when forty is NaN.

  c) compute_composite_scores(df: pd.DataFrame) -> pd.DataFrame
     - Add columns: height_inches (via parse_height_to_inches on 'ht'), speed_score, bmi (wt / height_inches^2), burst_score (vertical + broad_jump), catch_radius (height_inches — the proxy per user decision).
     - Let NaN propagate for missing raw measurables. Do NOT fill or impute.

  d) compute_position_percentiles(df: pd.DataFrame, score_cols: list) -> pd.DataFrame
     - For each col in score_cols, add "{col}_pos_pctl" using df.groupby("pos")[col].rank(pct=True).
     - rank(pct=True) already handles NaN correctly.

  e) build_jimmy_johnson_chart() -> dict
     - Hardcode picks 1-224 with exact values from RESEARCH.md Code Examples section.
     - Extend 225-262 via linear extrapolation: max(0.4, round(2.0 - (p - 224) * 0.042, 2)).
     - Return dict[int, float] mapping pick number to trade value.

  f) dedup_combine(combine_df: pd.DataFrame) -> pd.DataFrame
     - Separate rows with null pfr_id (keep all).
     - For non-null pfr_ids: sort by [pfr_id, _match_quality desc, season desc] where _match_quality = 1 if season == draft_year else 0.
     - drop_duplicates on pfr_id, keep="first". Drop _match_quality temp column.
     - pd.concat deduped non-nulls + null pfr_id rows, ignore_index=True.

  g) join_combine_draft(combine_df: pd.DataFrame, draft_df: pd.DataFrame) -> pd.DataFrame
     - Full outer merge: combine on "pfr_id", draft on "pfr_player_id", suffixes=("_combine", "_draft").
     - Coalesce overlapping columns (season, pos/position) preferring combine values, falling back to draft values.
     - Attach draft_value by mapping "pick" column through build_jimmy_johnson_chart().
     - Add has_pfr_id boolean column (True where pfr_id is not null).
     - Add gsis_id column from draft_df for downstream roster linkage.

  h) build_combine_draft_profiles(combine_df: pd.DataFrame, draft_df: pd.DataFrame) -> pd.DataFrame
     - Orchestrator: dedup_combine -> join_combine_draft -> compute_composite_scores -> compute_position_percentiles.
     - Position percentile cols: speed_score, bmi, burst_score, catch_radius.
     - Log at INFO: total rows, combine-only count, draft-only count, matched count, match rate %.
     - Log at WARNING: any unmatched players (pfr_ids that appear in only one source).
     - Assert final row count <= combine_deduped + draft_unique (no explosion).
     - Return final DataFrame.

  Use Google-style docstrings on all functions. Type hints on all parameters and returns.
