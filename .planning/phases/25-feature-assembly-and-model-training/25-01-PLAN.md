5. Create `src/feature_engineering.py` with these functions:

   - `_read_latest_local(subdir, season)` — reads latest parquet from `data/silver/{subdir}/season={season}/`, following the exact glob pattern from test_feature_vector.py.

   - `_read_bronze_schedules(season)` — reads latest parquet from `data/bronze/schedules/season={season}/`, filters to `game_type == "REG"`, returns columns: game_id, season, week, home_team, away_team, home_score, away_score, result, spread_line, total_line.

   - `assemble_game_features(season)` — Full assembly pipeline:
     a. Load game_context Silver for the season (has game_id, is_home, team, season, week)
     b. For each of the 8 SILVER_TEAM_LOCAL_DIRS sources, load and left-join on [team, season, week] (handle duplicate columns with suffixes, drop duplicates)
     c. Split into home (is_home==True) and away (is_home==False) DataFrames
     d. Join home and away on game_id with suffixes ("_home", "_away")
     e. Identify numeric columns (float64, int64) excluding identifiers (team, season, week, game_id, game_type, is_home)
     f. Compute differential: `diff_{col} = {col}_home - {col}_away` for each numeric column
     g. Add non-differential context: is_dome_home, rest_advantage_home, rest_advantage_away, div_game (inferred from game_id pattern or teams in same division)
     h. Join Bronze schedules on game_id to add labels: spread_line, total_line, home_score, away_score
     i. Compute `actual_margin = home_score - away_score` and `actual_total = home_score + away_score`
     j. Filter to game_type=="REG" only
     k. Fill wins/losses with 0 where NaN (early season)
     l. Return the assembled DataFrame

   - `get_feature_columns(game_df)` — Returns list of column names that are valid features (all diff_* columns plus non-differential context columns). Explicitly excludes all LABEL_COLUMNS plus identifiers (game_id, season, week, team_home, team_away, game_type).

   - `assemble_multiyear_features(seasons)` — Calls assemble_game_features for each season, concatenates, returns combined DataFrame. This is what the training pipeline will call.
