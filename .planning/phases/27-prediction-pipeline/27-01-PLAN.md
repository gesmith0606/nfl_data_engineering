GREEN phase -- Implement in scripts/generate_predictions.py:

Create the script with these components:

```python
#!/usr/bin/env python3
"""Generate weekly NFL game predictions with edge detection vs Vegas lines.

Edge convention:
  spread_edge = model_spread - vegas_spread
    Positive = model sees MORE home advantage than Vegas
  total_edge = model_total - vegas_total
    Positive = model expects HIGHER scoring than Vegas

Confidence tiers (per D-05):
  High:   |edge| >= 3.0 points
  Medium: 1.5 <= |edge| < 3.0 points
  Low:    |edge| < 1.5 points
"""
```

a) `classify_tier(edge_value)` function:
   - If pd.isna(edge_value): return None
   - If abs(edge_value) >= 3.0: return "high"
   - If abs(edge_value) >= 1.5: return "medium"
   - Else: return "low"

b) `generate_week_predictions(game_df, week, spread_model, spread_meta, total_model, total_meta)` function:
   - Filter game_df to requested week: `week_df = game_df[game_df["week"] == week].copy()`
   - If week_df is empty, return empty DataFrame
   - Get spread features from spread_meta["feature_names"], filter to available columns in week_df, warn on missing
   - Get total features from total_meta["feature_names"], filter to available columns in week_df, warn on missing
   - Fill NaN features with 0.0 (early season handling per FEAT-04)
   - Predict: `week_df["model_spread"] = spread_model.predict(week_df[available_spread])`
   - Predict: `week_df["model_total"] = total_model.predict(week_df[available_total])`
   - Rename Vegas columns: `week_df["vegas_spread"] = week_df["spread_line"]`, `week_df["vegas_total"] = week_df["total_line"]`
   - Compute edges: `week_df["spread_edge"] = week_df["model_spread"] - week_df["vegas_spread"]`, same for total
   - Classify tiers: `week_df["spread_confidence_tier"] = week_df["spread_edge"].abs().apply(classify_tier)`, same for total
   - Add metadata: `week_df["model_version"] = "v1.4.0"`, `week_df["prediction_timestamp"] = datetime.utcnow()`
   - Compute sort key: `week_df["_sort_key"] = week_df[["spread_edge", "total_edge"]].abs().max(axis=1)`
   - Sort: `week_df = week_df.sort_values("_sort_key", ascending=False, na_position="last")`
   - Select output columns: game_id, season, week, home_team, away_team, model_spread, model_total, vegas_spread, vegas_total, spread_edge, total_edge, spread_confidence_tier, total_confidence_tier, model_version, prediction_timestamp
   - Return output DataFrame

c) CLI `main()` with argparse:
   - `--season` (int, required)
   - `--week` (int, required)
   - `--model-dir` (str, default=None, overrides MODEL_DIR)
   - Load models with error handling (catch FileNotFoundError, print helpful message pointing to train_prediction_model.py)
   - Call `assemble_game_features(season)`, check non-empty
   - Call `generate_week_predictions(...)`
   - Print formatted console table (D-01, D-02, D-03)
   - Save Gold Parquet (D-04) to `data/gold/predictions/season={season}/week={week:02d}/predictions_{timestamp}.parquet`
   - Print summary: "N games, X high-confidence edges"
