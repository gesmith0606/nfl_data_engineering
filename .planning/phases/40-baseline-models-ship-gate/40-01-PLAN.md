Build the core player model training module that trains per-position, per-stat XGBoost models with walk-forward CV, SHAP-based per-group feature selection, and model serialization.

Output: `src/player_model_training.py` with all training functions, `tests/test_player_model_training.py` with unit tests covering model count, walk-forward folds, feature selection groups, and stat-to-fantasy conversion.
