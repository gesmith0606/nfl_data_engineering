    2. Constants: PROJECT_ROOT, BRONZE_DIR, SILVER_DIR (same as silver_team_transformation.py).

    3. Team abbreviation normalization map:
       ```python
       TEAM_ABBR_NORM = {"LAR": "LA", "WSH": "WAS"}
       ```

    4. Helper `_read_local_bronze(subdir, season)`: Read latest parquet from BRONZE_DIR/subdir/season=YYYY/*.parquet. Return empty DataFrame if no files found.

    5. Helper `_normalize_team(df, col="team")`: Apply TEAM_ABBR_NORM to a team column. Return df with normalized teams.

    6. Helper `_normalize_name(df, col)`: Apply .str.strip().str.lower() to a name column for join matching. Return df with normalized name column added as `{col}_norm`.

    7. Helper `_save_local_silver(df, key, ts)`: Save parquet to local Silver dir (same pattern as team script).

    8. Core function `process_season(season)`:
       a. Read master player roster from `data/bronze/players/weekly/season={season}/*.parquet`. If empty, log error and return None. Keep columns: player_id (as player_gsis_id), player_display_name, position, recent_team, season, week. This is the left side of all joins.
       b. Normalize master: `_normalize_name(master, "player_display_name")` and `_normalize_team(master, "recent_team")`.
       c. Read NGS receiving/passing/rushing from Bronze. For each, if empty log warning. If not empty:
          - Filter week > 0 (inside compute functions, but verify)
          - NGS has player_gsis_id -- join directly on [player_gsis_id, season, week]
          - Call compute_ngs_receiving_profile, compute_ngs_passing_profile, compute_ngs_rushing_profile
          - Left-merge results onto master by [player_gsis_id, season, week]
       d. Read PFR weekly pass from Bronze. If not empty:
          - Normalize name (player column) and team
          - Call compute_pfr_pressure_rate
          - Left-merge onto master by [player_display_name_norm, recent_team_norm, season, week] (using normalized name+team)
       e. Read PFR weekly def from Bronze. If not empty:
          - Call compute_pfr_team_blitz_rate (produces team-level rows)
          - Left-merge onto master by [recent_team, season, week] (team-level join, so every player on that team gets the same values)
       f. Read QBR from Bronze (look for qbr_weekly_* files). If not empty:
          - Normalize name (name_display column) and team (team_abb column)
          - Call compute_qbr_profile
          - Left-merge onto master by [player_display_name_norm, recent_team_norm, season, week]
          - After merge, set QBR columns to NaN for non-QB rows: `df.loc[df["position"] != "QB", qbr_cols] = np.nan`
       g. Log NaN coverage for all advanced stat columns via log_nan_coverage.
       h. Record row count: assert output rows == master rows (no drops, no explosion).
       i. Return merged DataFrame.

    9. Main function `main()`:
       - argparse with `--seasons` (nargs="+", type=int, default=PLAYER_DATA_SEASONS)
       - For each season: call process_season, if result is not None, save via _save_local_silver using SILVER_PLAYER_S3_KEYS["advanced_profiles"] pattern
       - Print summary: seasons processed, rows per season, any warnings

    10. if __name__ == "__main__": main()

    - Never fail on match quality -- warnings only
    - PFR match rate logged at WARNING (unmatched names) and INFO (overall rate)
    - Use .str.strip().str.lower() for name normalization (not fuzzy matching per RESEARCH.md)
