Create `scripts/train_ensemble.py` following the existing CLI pattern from `train_prediction_model.py`.

**Argparse flags** (per D-09):
- `--tune` (store_true): run Optuna tuning with 50 trials per model (per D-07). Without this flag, use conservative defaults for quick iteration.
- `--trials` (int, default=50): number of Optuna trials per model per target (per D-07)
- `--ensemble-dir` (str, default=None): output directory, defaults to ENSEMBLE_DIR from config

**Main workflow:**
1. Import from ensemble_training, feature_engineering, config
2. Assemble features: `all_data = assemble_multiyear_features()`
3. Determine feature columns:
   ```python
   from config import SELECTED_FEATURES
   if SELECTED_FEATURES is not None:
       feature_cols = [c for c in SELECTED_FEATURES if c in all_data.columns]
       print(f"Using {len(feature_cols)} selected features from config")
   else:
       feature_cols = get_feature_columns(all_data)
       print(f"WARNING: SELECTED_FEATURES is None, using all {len(feature_cols)} features")
   ```
4. If `--tune`:
   - For each model type (xgb, lgb, cb) and each target (spread, total):
     - Create Optuna study with `direction="minimize"`
     - Run `study.optimize(objective, n_trials=args.trials)`
     - Collect best_params per model
   - Optuna objectives use `walk_forward_cv_with_oof` mean_mae (per D-07)
   - XGB search space: max_depth [2,6], learning_rate [0.01,0.15 log], min_child_weight [3,20], subsample [0.6,0.9], colsample_bytree [0.5,0.8], reg_alpha [0.1,10 log], reg_lambda [1,20 log]
   - LGB search space: max_depth [2,6], learning_rate [0.01,0.15 log], min_child_samples [10,50], subsample [0.6,0.9], colsample_bytree [0.5,0.8], reg_alpha [0.1,10 log], reg_lambda [1,20 log]
   - CB search space: depth [2,6], learning_rate [0.01,0.15 log], l2_leaf_reg [1,20 log], min_data_in_leaf [10,50], subsample [0.6,0.9], rsm [0.5,0.8]
5. Call `train_ensemble(all_data, feature_cols, xgb_params, lgb_params, cb_params, ensemble_dir)`
6. Print summary: per-model CV MAE, Ridge coefficients, selected alpha, total training time
7. Return exit code 0 on success, 1 on error

**Optuna integration detail:**
- Set `optuna.logging.set_verbosity(optuna.logging.WARNING)` to reduce noise
- Use `study_name=f"{model_type}_{target}"` for each study
- Print best params and MAE after each study completes
