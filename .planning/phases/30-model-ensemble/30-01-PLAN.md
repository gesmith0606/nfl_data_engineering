A. **Model factory functions** (per D-04, D-08):
- `make_xgb_model(params: dict) -> xgb.XGBRegressor` -- extracts early_stopping_rounds from params, passes rest to constructor

B. **Generalized walk-forward CV with OOF** (per D-01, D-05):
```python
def walk_forward_cv_with_oof(
    all_data: pd.DataFrame,
    feature_cols: list,
    target_col: str,
    model_factory: Callable[[], Any],
    fit_kwargs_fn: Optional[Callable] = None,
    val_seasons: Optional[list] = None,
) -> Tuple[WalkForwardResult, pd.DataFrame]:
```
- Iterates VALIDATION_SEASONS (default [2019..2023]), per D-05
- For each val_season: train on seasons < val_season, validate on val_season
- Raises ValueError if val_season == HOLDOUT_SEASON
- Creates fresh model via model_factory() each fold
- For XGBoost: model.fit(X_train, y_train, eval_set=[(X_val, y_val)], verbose=False)
- For LightGBM: model.fit(X_train, y_train, eval_set=[(X_val, y_val)], callbacks=[lgb.early_stopping(50, verbose=False)])
- For CatBoost: model.fit(X_train, y_train, eval_set=(X_val, y_val))
- The fit_kwargs_fn is a callable that receives (X_train, y_train, X_val, y_val) and returns a dict of extra kwargs for .fit(). This handles API differences:
  - XGBoost: returns {"eval_set": [(X_val, y_val)], "verbose": False}
  - LightGBM: returns {"eval_set": [(X_val, y_val)], "callbacks": [lgb.early_stopping(50, verbose=False)]}
  - CatBoost: returns {"eval_set": (X_val, y_val)}
- Collects OOF predictions into a DataFrame with columns: game_id, season, oof_prediction
- Returns Tuple[WalkForwardResult, oof_df]

```python
def assemble_oof_matrix(
```

E. **Full ensemble training** (per D-02, D-10, D-11):
```python
def train_ensemble(
    all_data: pd.DataFrame,
    feature_cols: list,
    xgb_params: Optional[dict] = None,
    lgb_params: Optional[dict] = None,
    cb_params: Optional[dict] = None,
    ensemble_dir: Optional[str] = None,
) -> dict:
```
- For each target in ["spread", "total"]:
  - target_col = "actual_margin" for spread, "actual_total" for total
  - Run walk_forward_cv_with_oof for XGB, LGB, CB
  - Assemble OOF matrix
  - Train Ridge meta-learner
  - Train final models on all data < HOLDOUT_SEASON (last val season as eval_set for early stopping)
- Save all artifacts per D-10: xgb_spread.json, lgb_spread.txt, cb_spread.cbm, ridge_spread.pkl, etc.
- Save metadata.json per D-11 with ensemble_version, trained_at, training_seasons, holdout_season, selected_features, n_features, and per-target base model CV MAEs, Ridge alpha and coefficients
- Returns metadata dict

