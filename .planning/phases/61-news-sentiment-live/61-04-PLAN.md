| Boundary | Description |
|----------|-------------|
| github-actions → local runner | Secrets (ANTHROPIC_API_KEY) cross into the workflow env; must never echo them |
| runner → git push | The workflow pushes data commits to main; limited to bot user and paths |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-61-04-01 | Information Disclosure | GHA log output | mitigate | Never `echo $ANTHROPIC_API_KEY`; only log `ENABLE_LLM_ENRICHMENT` which is non-secret |
| T-61-04-02 | Denial of Service | Upstream RSS 5xx | mitigate | Per-step try/except in _run_* helpers; pipeline continues when one source fails |
| T-61-04-03 | Tampering | Bot-pushed commits | accept | Commits restricted to `data/bronze/sentiment/**` and related dirs; diff reviewable; user can revert |
| T-61-04-04 | Repudiation | Silent LLM fallback | mitigate | Step 6 logs explicitly when rule-only path is taken (D-06 log line) |
