  <action>
    1. Add test classes TestPace, TestPROE, TestFourthDown, TestEarlyDownRunRate to tests/test_team_analytics.py. Run tests (RED).

    2. In `src/team_analytics.py`, add these functions:

    `compute_pace(valid_plays: pd.DataFrame) -> pd.DataFrame`:
    - Group by (posteam, season, week), count rows as pace. Rename posteam->team.
    - Return DataFrame with columns: team, season, week, pace.

    `compute_proe(valid_plays: pd.DataFrame) -> pd.DataFrame`:
    - Group by (posteam, season, week):
      - total_plays: count of all rows
      - pass_plays: sum of pass_attempt
      - mean_xpass: mean of xpass (dropna handled by pandas mean)
    - actual_pass_rate = pass_plays / total_plays
    - proe = actual_pass_rate - mean_xpass
    - Rename posteam->team. Return team, season, week, proe.

    `compute_fourth_down_aggressiveness(pbp_df: pd.DataFrame) -> pd.DataFrame`:
    - NOTE: Takes raw (unfiltered) PBP or at least PBP that includes punt/field_goal plays on 4th down. The caller should pass _filter_valid_plays output PLUS 4th down punt/FG plays, OR this function should filter from the raw PBP itself.
    - Best approach: accept raw PBP, apply own filtering: season_type=='REG', week<=18, down==4, play_type in ['pass','run','punt','field_goal'], exclude no_play.
    - go_attempts = play_type in ['pass', 'run'] on 4th down
    - total_decisions = play_type in ['pass', 'run', 'punt', 'field_goal'] on 4th down
    - go_rate = go_attempts / total_decisions per (posteam, season, week)
    - fourth_down_success_rate: on go attempts only, sum(fourth_down_converted) / (sum(fourth_down_converted) + sum(fourth_down_failed)). Handle zero go attempts as NaN.
    - Rename posteam->team. Return team, season, week, fourth_down_go_rate, fourth_down_success_rate.

    `compute_early_down_run_rate(valid_plays: pd.DataFrame) -> pd.DataFrame`:
    - Filter to down <= 2.
    - Group by (posteam, season, week):
      - early_down_run_rate = sum(rush_attempt) / count of rows
    - Rename posteam->team. Return team, season, week, early_down_run_rate.

    3. Add `compute_tendency_metrics(pbp_df: pd.DataFrame) -> pd.DataFrame` orchestrator:
    - valid = _filter_valid_plays(pbp_df) -- for pace, PROE, early-down
    - Compute pace, PROE, early-down run rate from valid plays
    - Compute 4th down from raw pbp_df (needs punt/FG plays)
    - Merge all on (team, season, week)
    - apply_team_rolling on all stat columns
    - Return final DataFrame

    4. Run tests (GREEN). Run full suite.
