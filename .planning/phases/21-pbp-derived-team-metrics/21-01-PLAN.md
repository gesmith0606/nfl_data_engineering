  <action>
Add a new section in `src/team_analytics.py` after the existing "Tendency Metric Functions" section, titled "PBP-Derived Metric Functions (Phase 21)". Add these functions in order:

1. **`_filter_st_plays(pbp_df)`** -- helper function. Filter to `special_teams_play == 1` OR `play_type.isin(['field_goal', 'punt', 'kickoff', 'extra_point'])`. Apply `season_type == 'REG'` and `week <= 18` guards (same pattern as `compute_fourth_down_aggressiveness`). Return filtered DataFrame.

2. **`compute_penalty_metrics(pbp_df)`** -- receives raw PBP. Filter `penalty == 1`, drop rows where `penalty_team` is null. Group offensive penalties (`penalty_team == posteam`) by `(posteam, season, week)` to get `off_penalties` (count) and `off_penalty_yards` (sum of `penalty_yards`). Group defensive penalties (`penalty_team == defteam`) by `(defteam, season, week)` for `def_penalties` and `def_penalty_yards`. Merge on `(team, season, week)` with `how='outer'`. Fill NaN counts with 0.

3. **`compute_opp_drawn_penalties(pbp_df)`** -- receives raw PBP. Same filter as penalty metrics. "Drawn" = penalties committed by the opponent. Offensive penalties drawn = defensive penalties grouped by `posteam` (offense benefits from opponent's defensive penalties). Columns: `off_penalties_drawn`, `off_penalty_yards_drawn`, `def_penalties_drawn`, `def_penalty_yards_drawn`. Merge on `(team, season, week)`.

4. **`compute_turnover_luck(pbp_df)`** -- receives raw PBP. Filter `season_type=='REG'`, `week<=18`, `fumble == 1`. For each `(posteam, season, week)`, count `fumbles_lost` (where `fumble_recovery_1_team != posteam`) and total fumbles. Compute `own_fumble_recovery_rate` = recovered / total. For defense: count `fumbles_forced` grouped by `defteam`. Compute `opp_fumble_recovery_rate`. Merge on `(team, season, week)`. Then compute season-to-date expanding recovery rate with `groupby(['team','season']).expanding().mean()` after `shift(1)`. Add `is_turnover_lucky` = 1 when own_fumble_recovery_rate > 0.60, -1 when < 0.40, else 0 (using `np.where`).

5. **`compute_red_zone_trips(valid_plays)`** -- receives filtered valid plays. Filter `yardline_100 <= 20`. Group by `(posteam, season, week)` and count `drive` nunique as `off_rz_trips`. Group by `(defteam, season, week)` for `def_rz_trips`. Merge on `(team, season, week)`.

6. **`compute_third_down_rates(valid_plays)`** -- receives filtered valid plays. Filter `down == 3`. Group by `(posteam, season, week)`: `off_third_down_rate = third_down_converted.sum() / (third_down_converted.sum() + third_down_failed.sum())`. Same for `(defteam, season, week)` as `def_third_down_rate`. Handle division by zero with `np.where`.

7. **`compute_explosive_plays(valid_plays)`** -- receives filtered valid plays. Explosive pass = `play_type == 'pass'` AND `yards_gained >= 20`. Explosive rush = `play_type == 'run'` AND `yards_gained >= 10`. For offense, group by `(posteam, season, week)`: `off_explosive_pass_rate = explosive_passes / total_passes`, `off_explosive_rush_rate = explosive_rushes / total_rushes`. Same for defense by `(defteam, season, week)`. Merge on `(team, season, week)`.

8. **`compute_sack_rates(valid_plays)`** -- receives filtered valid plays. Offensive sack rate: group by `(posteam, season, week)`, compute `off_sack_rate = sack.sum() / pass_attempt.sum()` (sacks / dropbacks). Defensive sack rate: group by `(defteam, season, week)`, `def_sack_rate = sack.sum() / pass_attempt.sum()`. Note: `pass_attempt` includes sacks in the PBP data. Merge on `(team, season, week)`.

All functions must:
- Have Google-style docstrings with Args/Returns
- Include type hints (pd.DataFrame -> pd.DataFrame)
- Return empty DataFrame with correct column names when no data found
- Use logger.info for row count reporting
