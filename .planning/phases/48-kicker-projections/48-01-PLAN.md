# Phase 48-01: Kicker Projection Model - Plan

**Created:** 2026-04-02
**Status:** Ready

## Tasks

### T-01: Kicker Feature Engineering
- Create `src/kicker_analytics.py`
- From PBP: extract per-kicker per-week stats
  - FG attempts/makes by distance bucket (< 40, 40-49, 50+)
  - XP attempts/makes
  - Fantasy points (3 per FG, 1 per XP; 5 for 50+ in some formats)
- Team-level features:
  - Red zone stall rate: (RZ drives - RZ TDs) / RZ drives (rolling 3 games, shift(1))
  - Drives per game reaching FG range (inside 40)
  - Offensive efficiency (yards per drive) vs RZ conversion rate mismatch
- Opposing defense features:
  - Opponent RZ defense quality: RZ TD rate allowed (rolling 3 games)
  - Opponent drives allowed into FG range

### T-02: Game Script + Venue Features
- From game prediction model: use predicted spread for game script
  - Close game probability: |predicted_spread| < 7
  - Blowout probability: |predicted_spread| > 14 (fewer FG opportunities)
- From game_context Silver:
  - Dome indicator (higher accuracy)
  - Wind speed (accuracy penalty for outdoor 40+ yarders)
  - Altitude (Denver bonus for long FGs)
  - Temperature (cold weather accuracy penalty)

### T-03: Kicker Projection Engine
- Add kicker projection function to `src/projection_engine.py` or create `src/kicker_projection.py`
- Baseline: team FG attempts (from red zone stall rate) × kicker accuracy × distance distribution
- Multipliers: game script (close game bonus), venue/weather, opponent RZ defense
- Output: projected FG makes, projected XP makes, projected fantasy points, floor, ceiling

### T-04: Integration
- Add 'K' to projection output in generate_projections.py
- Add kicker to draft optimizer (VORP baseline: K=13 or similar)
- Add kicker to scoring_calculator.py if not already there
- Wire into backtest_projections.py

### T-05: Tests + Backtest
- Create tests/test_kicker_projections.py
- Unit tests for red zone stall rate, distance bucketing, game script multiplier
- Backtest kicker projections against 2022-2024 actuals
- Report MAE for kicker position
