1. Create `scripts/train_prediction_model.py`:

   **Header and path setup** (follow existing scripts pattern):
   ```python
   #!/usr/bin/env python3
   """Train XGBoost game prediction models with walk-forward CV and Optuna tuning.

   Usage:
       python scripts/train_prediction_model.py --target spread
       python scripts/train_prediction_model.py --target total --trials 100
       python scripts/train_prediction_model.py --target spread --no-tune
   """
   import argparse
   import os
   import sys
   sys.path.insert(0, os.path.join(os.path.dirname(__file__), ".."))
   ```

   **argparse arguments:**
   - `--target` (required, choices=["spread", "total"]) — "spread" trains on actual_margin, "total" trains on actual_total
   - `--trials` (default=50, type=int) — number of Optuna trials
   - `--no-tune` (store_true) — skip Optuna, use CONSERVATIVE_PARAMS directly
   - `--seasons` (nargs="+", type=int, default=None) — override TRAINING_SEASONS if desired

   **Target column mapping:**
   ```python
   TARGET_MAP = {
       "spread": "actual_margin",
       "total": "actual_total",
   }
   ```

   **Main flow:**
   a. Parse args
   b. Print banner: "Training {target} prediction model..."
   c. Load data: `all_data = assemble_multiyear_features(seasons)` where seasons = args.seasons or TRAINING_SEASONS
   d. Get feature columns: `feature_cols = get_feature_columns(all_data)`
   e. Print data stats: f"{len(all_data)} games, {len(feature_cols)} features, seasons {min(seasons)}-{max(seasons)}"
   f. If `--no-tune`:
      - Use CONSERVATIVE_PARAMS
      - Run walk_forward_cv to get CV scores
      - Print CV results
   g. Else (Optuna tuning):
      - Create Optuna study with `direction="minimize"` and `study_name=f"{target}_prediction"`
      - Suppress Optuna INFO logs: `optuna.logging.set_verbosity(optuna.logging.WARNING)`
      - Define objective function that:
        - Suggests params using the search ranges from RESEARCH.md
        - Adds fixed params: n_estimators=500, early_stopping_rounds=50, objective="reg:squarederror", random_state=42, verbosity=0
        - Calls walk_forward_cv with suggested params
        - Returns result.mean_mae
      - Run `study.optimize(objective, n_trials=args.trials)`
      - Print best trial: f"Best MAE: {study.best_trial.value:.4f}"
      - Print best params
      - Use study.best_params merged with fixed params as final params
   h. Train final model: `train_final_model(all_data, feature_cols, target_col, params=best_params, target_name=args.target)`
   i. Feature importance report:
      - Load the just-trained model
      - Create DataFrame with feature names and `model.feature_importances_` (gain-based)
      - Sort descending by importance
      - Print "Top 20 Features by Gain:" followed by top 20 rows
      - Save full importance to `models/{target}/feature_importance.csv`
   j. Print summary: model path, metadata path, CV MAE, number of features
