1. Create `src/model_training.py` with these components:

   **Data class / named tuple:**
   ```python
   from dataclasses import dataclass, field
   from typing import List, Dict, Any, Optional

   @dataclass
   class WalkForwardResult:
       mean_mae: float
       fold_maes: List[float]
       fold_details: List[Dict[str, Any]]  # {train_seasons, val_season, train_size, val_size, mae}
   ```

   **walk_forward_cv(all_data, feature_cols, target_col, params=None, val_seasons=None):**
   - `params` defaults to CONSERVATIVE_PARAMS from config
   - `val_seasons` defaults to VALIDATION_SEASONS from config ([2019, 2020, 2021, 2022, 2023])
   - For each val_season: train = all_data[all_data["season"] < val_season], val = all_data[all_data["season"] == val_season]
   - Assert val_season != HOLDOUT_SEASON (2024) to prevent accidental leakage
   - Create XGBRegressor with params. Set early_stopping_rounds from params (pop it from fit params, pass to fit() via eval_set)
   - fit with eval_set=[(val_X, val_y)] for early stopping
   - Compute MAE via sklearn.metrics.mean_absolute_error
   - Return WalkForwardResult with mean of fold MAEs and per-fold details

   **train_final_model(all_data, feature_cols, target_col, params=None, target_name=None):**
   - `target_name` is "spread" or "total" — determines output subdirectory
   - Trains on ALL data where season < HOLDOUT_SEASON (2016-2023)
   - Uses last validation season (2023) as eval_set for early stopping
   - Saves model via `model.save_model(os.path.join(MODEL_DIR, target_name, "model.json"))`
   - Saves metadata.json alongside with:
     - target, training_seasons (list), n_features (int), feature_names (list)
     - cv_scores (dict with mean_mae and fold_maes from walk_forward_cv)
     - best_params (dict), trained_at (ISO timestamp)
   - Creates MODEL_DIR/{target_name}/ directory if it doesn't exist (os.makedirs)
   - Returns the trained model and metadata dict

   **load_model(target_name):**
   - Loads model from `models/{target_name}/model.json` via `xgb.XGBRegressor(); model.load_model(path)`
   - Loads metadata from `models/{target_name}/metadata.json`
   - Returns (model, metadata) tuple

   Import pattern:
   ```python
   import xgboost as xgb
   from sklearn.metrics import mean_absolute_error
   from src.config import CONSERVATIVE_PARAMS, TRAINING_SEASONS, VALIDATION_SEASONS, HOLDOUT_SEASON, MODEL_DIR, LABEL_COLUMNS
   from src.feature_engineering import assemble_multiyear_features, get_feature_columns
   ```
