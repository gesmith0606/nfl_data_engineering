### Interface contract (downstream plans depend on these exact names)

CSS custom properties (in `tokens.css`):
```
/* Typography scale — 7 steps, mobile-first */
--fs-xs:   0.75rem;   --lh-xs: 1.0rem;
--fs-sm:   0.875rem;  --lh-sm: 1.25rem;
--fs-body: 1rem;      --lh-body: 1.5rem;
--fs-lg:   1.125rem;  --lh-lg: 1.625rem;
--fs-h3:   1.25rem;   --lh-h3: 1.75rem;
--fs-h2:   1.5rem;    --lh-h2: 2rem;
--fs-h1:   2rem;      --lh-h1: 2.25rem;

/* Spacing (4px grid) */
--space-1:  0.25rem;  --space-2:  0.5rem;   --space-3:  0.75rem;
--space-4:  1rem;     --space-5:  1.25rem;  --space-6:  1.5rem;
--space-8:  2rem;     --space-10: 2.5rem;   --space-12: 3rem;

/* Motion durations — Emil Kowalski bands */
--motion-instant: 100ms;
--motion-fast:    150ms;
--motion-base:    220ms;
--motion-slow:    320ms;
--motion-slower:  480ms;

/* Motion easings */
--ease-out-standard: cubic-bezier(0.2, 0, 0, 1);
--ease-in-out-standard: cubic-bezier(0.4, 0, 0.2, 1);
--ease-spring: linear(0, 0.0019, 0.0077, ...)  /* or keep cubic-bezier fallback */

/* Elevation scale — lean on existing theme --shadow-* but expose semantic names */
--elevation-flat:   var(--shadow-2xs);
--elevation-raised: var(--shadow-sm);
--elevation-overlay:var(--shadow-md);
--elevation-modal:  var(--shadow-xl);
```

TypeScript mirror (in `design-tokens.ts`):
```ts
export const MOTION = {
  instant: 0.1,
  fast: 0.15,
  base: 0.22,
  slow: 0.32,
  slower: 0.48,
} as const;

export const EASE = {
  outStandard: [0.2, 0, 0, 1] as const,
  inOutStandard: [0.4, 0, 0.2, 1] as const,
};

export const TYPE_SCALE = ['xs','sm','body','lg','h3','h2','h1'] as const;
export type TypeScale = typeof TYPE_SCALE[number];

export const SPACE = [1,2,3,4,5,6,8,10,12] as const;
```

Note: exact scale values are defaults — the design-engineer may tune them based on AUDIT-BASELINE.md typography drift findings. But names and contracts above MUST NOT change since downstream plans reference them.
