Return dict keyed by data type with aggregated metrics: file_count, total_size_mb, season_range (min-max from `season=YYYY` partition dirs), column_count (from first file's schema), last_modified.

    - `main()`: argparse CLI with: `--base-dir` (default: data/bronze), `--output` (default: stdout, or path to write markdown file), `--s3` flag (use scan_s3 instead of scan_local).

    **Per user decision:** No row counts (too slow). Metrics: file count, total size (MB), season range, column count, last ingestion date.
