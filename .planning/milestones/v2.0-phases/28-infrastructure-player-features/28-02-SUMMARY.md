## Decisions Made

- carry_share is computed from carries/team_total_carries because Bronze weekly data does not include this column
- backup_qb_start boolean is excluded from the feature set (no _roll3/_roll6/_std suffix, not in _PRE_GAME_CONTEXT) -- the rolling columns (qb_passing_epa_roll3 etc.) carry the signal; boolean can be added later if needed

- Top-2 RBs by carries and top-3 WR/TEs by target share for positional quality (matches research recommendations)
