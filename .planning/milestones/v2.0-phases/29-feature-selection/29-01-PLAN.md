       - `selected_features: List[str]`
       - `dropped_correlation: Dict[str, str]`  (dropped_feat -> kept_feat)
       - `dropped_low_importance: List[str]`
       - `shap_scores: Dict[str, float]`  (feature -> mean |SHAP|)
       - `correlated_pairs: List[Tuple[str, str, float]]`  (feat_a, feat_b, r)

    3. `filter_correlated_features(data, feature_cols, shap_rank, threshold=0.90)`:
       - Compute `data[feature_cols].corr(method="pearson").abs()`
       - Extract upper triangle pairs above threshold, sort descending by correlation
       - For each pair: skip if either already dropped, else drop the one with lower `shap_rank` value (per D-03)
       - Return `(surviving_features, dropped_map, correlated_pairs_list)`

    4. `select_features_for_fold(train_data, feature_cols, target_col, target_count, correlation_threshold=0.90, params=None)`:
       - Call `_assert_no_holdout(train_data, "feature selection")` (FSEL-04, D-08)
       - Drop features with zero variance in `train_data` (handle Phase 28 features absent in early seasons)

For eval_set, use a 20% random split of train_data (not temporal - this is just for early stopping in the quick SHAP model).
       - Compute SHAP: `shap.TreeExplainer(model).shap_values(X_train_sample)` where sample is min(500, len(train_data)) rows for speed. `mean_abs_shap = np.mean(np.abs(shap_values), axis=0)`.
       - Call `filter_correlated_features(train_data, active_features, shap_rank, correlation_threshold)`
       - Truncate surviving features to `target_count` by SHAP rank (highest first)
       - Record `fold_seasons = sorted(train_data["season"].unique().tolist())` (D-09)
       - Return `FeatureSelectionResult(...)` with all fields populated
