## Milestone Summary

**Key Decisions:**

- Design consolidation option-a: impeccable as primary greenfield generator; redesign-skill + emil-design-eng as specialized (existing-code / advisory); taste-skill + soft-skill aliased without standalone firing. Per-role Invocation Routing blocks in all 5 SKILL.md files plus design-engineer.md.
- Rule-first sentiment architecture locked (D-01, D-04, D-06): Haiku enrichment demoted to strictly optional website-only path; daily cron never blocked by upstream flakes (try/except → warning + continue on every source).
- CI quality-gate keys exclusively on sanity-check exit code (0=deploy, 1=block); no stdout grep — preserves warnings-allowed contract.
- Design-token layer is additive-only (tokens.css is `:root` custom properties; theme.css keeps sole ownership of colors; zero visual change on shipment).
- ADVR-02 ships `meta.data_as_of` + `/api/projections/latest-week` + advisor auto-resolve, closing the "top-10 RBs" query loop without client-side assumptions.
- Matchup advantages cite raw silver positional rank (`#N/32 vs POS`) instead of synthesized scores — opaque calculations replaced with auditable primary source.

**Issues Resolved:**

- Stale rosters (2026 trades/FA): Davante Adams NYJ→LA, Puka Nacua LAR→LA, daily Sleeper refresh updates both team AND position.
- CI pipeline lacked projection-quality gate before deploys — quality-gate job now blocks Vercel + Railway/ECR deploys on CRITICAL issues.
- Advisor FAIL count: 5 → 0 on live Railway (7 PASS / 5 WARN / 0 FAIL).
- Matchup view used placeholder hash-based defensive rosters — replaced with real NFL names + injury badges.
- 5 overlapping design skills could fire simultaneously and produce incoherent output — consolidated routing prevents multi-invocation.

**Issues Deferred:**

- DQAL-03 warning count >10 (34 warnings): 7 negative projections (clamp bug in projection_engine.py), 5 missing 2025 rookies (requires draft-class ingestion), 18 rank-gap threshold calibration, 2 stale Silver (resolves on next weekly pipeline). Tracked as known gap — CI gate delivers full security value without this fix.
- 61-03 event-based projection adjustment: opt-in behind `--use-events` but ship gate not cleared (backtest structurally null — 0/48 weeks had Gold events data). Feature ships dormant pending Bronze event accumulation.
- 61-HUMAN-UAT remaining concerns all resolved via live endpoint checks at milestone close (2026-04-20 21:28 UTC).

**Technical Debt Incurred:**

- Negative projection clamp bug surfaces as 7 sanity-check warnings. Isolated to bench fringe players; does not affect top-10 display.
