epa          -- expected points added per play (NaN ~1.2% on pass+run)
success      -- binary 0/1 per play
cpoe         -- completion probability over expected (NaN ~11.3% on pass plays)
play_type    -- 'pass', 'run', 'punt', 'field_goal', 'no_play', etc.
posteam      -- possessing team abbreviation
defteam      -- defending team abbreviation
yardline_100 -- yards from opponent end zone (<=20 = red zone)
drive        -- drive number within game
touchdown    -- binary 0/1
pass_attempt -- binary 0/1
season, week, season_type

    1. Create `tests/test_team_analytics.py` with test classes TestEPA, TestSuccessRate, TestCPOE, TestRedZone using synthetic PBP DataFrames. Write all tests first (RED).

    2. In `src/team_analytics.py`, add these functions below the existing skeleton:

    `compute_team_epa(valid_plays: pd.DataFrame) -> pd.DataFrame`:
    - Group by (posteam, season, week) for offense: agg epa mean as off_epa_per_play. For pass/rush splits, filter play_type=='pass' or 'run' within the group agg (use named aggregation or separate groupby-filter-merge). Rename posteam->team.
    - Group by (defteam, season, week) for defense: agg epa mean as def_epa_per_play. Rename defteam->team.
    - Merge offense and defense on (team, season, week) with outer join.

    `compute_team_success_rate(valid_plays: pd.DataFrame) -> pd.DataFrame`:
    - Group by (posteam, season, week): mean of success -> off_success_rate. Rename posteam->team.
    - Group by (defteam, season, week): mean of success -> def_success_rate. Rename defteam->team.
    - Merge on (team, season, week).

    `compute_team_cpoe(valid_plays: pd.DataFrame) -> pd.DataFrame`:
    - Filter to rows where cpoe is not null (only actual pass attempts with CPOE).
    - Group by (posteam, season, week): mean of cpoe -> off_cpoe. Rename posteam->team.
    - Note: CPOE is offense-only (no defense CPOE).

    `compute_red_zone_metrics(valid_plays: pd.DataFrame) -> pd.DataFrame`:
    - Filter to yardline_100 <= 20 for red zone plays.
    - Offense (group by posteam, season, week):
      - off_rz_epa: mean(epa)
      - off_rz_success_rate: mean(success)
      - off_rz_pass_rate: mean(pass_attempt) -- fraction of RZ plays that are passes
      - off_rz_tds: sum(touchdown)
      - off_rz_drives: nunique(drive) -- drive-based denominator
      - off_rz_td_rate: off_rz_tds / off_rz_drives
    - Defense (group by defteam, season, week): same metrics with def_ prefix.
    - Merge on (team, season, week). Teams with no RZ plays get NaN (natural from left join).
    - Drop intermediate columns (off_rz_tds, off_rz_drives, def_rz_tds, def_rz_drives).

    - Call _filter_valid_plays(pbp_df)
    - Call each of the 4 compute functions
    - Merge all results on (team, season, week)
    - Identify all stat columns (off_*, def_*, excluding team/season/week)
    - Call apply_team_rolling(merged, stat_cols)
    - Log summary: row count, team count, season range
    - Return final DataFrame
