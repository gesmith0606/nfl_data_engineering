5. **`parse_finnedai(json_path: str, seasons: list[int]) -> pd.DataFrame`** -- per research:
   - Load JSON with `json.load()`
   - Filter to requested seasons (cast season to int)
   - Drop entries where `str(entry.get("home_team", "")) == "0" or str(entry.get("away_team", "")) == "0"` -- log count
   - Create DataFrame, cast `season` to int
   - Parse date: `pd.to_datetime(df["date"].astype(int).astype(str), format="%Y%m%d").dt.date` -> column `gameday`
   - Map team names: `df["home_team_nfl"] = df["home_team"].map(FINNEDAI_TO_NFLVERSE)` and same for away
   - **D-11:** Explicitly preserve NaN for missing `home_open_spread` values -- do NOT fill with zero or drop rows where opening lines are missing. Use passthrough: leave the column as-is after DataFrame creation (pandas naturally represents missing JSON fields as NaN).
   - **D-12:** Exclude postponed/cancelled games: filter out entries where `home_final` is absent or NaN (no final score means no prediction target exists). Log count of excluded postponed games.
   - Return DataFrame

6. **`resolve_newyork(odds_df: pd.DataFrame, schedules: dict[int, pd.DataFrame]) -> pd.DataFrame`**:
   - Find rows where `home_team_nfl` or `away_team_nfl` is None (unmapped "NewYork")
   - For each such row: look up nflverse schedule for that season, find NYG and NYJ home games on that date, match by opponent to determine which NY team
   - Set the correct abbreviation (NYG or NYJ) on the row
   - If still ambiguous after opponent check, log warning but keep row with best guess
   - Return updated DataFrame

7. **`align_spreads(df: pd.DataFrame) -> pd.DataFrame`** -- per research Pattern 3:
   - `df["opening_spread"] = -df["home_open_spread"]` (negate: FinnedAI negative=home favored -> nflverse positive=home favored)
   - `df["closing_spread"] = -df["home_close_spread"]`
   - `df["opening_total"] = df["open_over_under"]`
   - `df["closing_total"] = df["close_over_under"]`
   - `df["home_moneyline"] = df["home_close_ml"]` (per D-16)
   - `df["away_moneyline"] = df["away_close_ml"]`
   - Return df

8. **`join_to_nflverse(odds_df: pd.DataFrame, season: int) -> pd.DataFrame`** -- per research Pattern 4:
   - `sched = nfl.import_schedules([season])`
   - Convert nflverse `gameday` to `datetime.date` for join
   - Merge on `(season, home_team_nfl, gameday)` matching `(season, home_team, gameday_date)` with `how="left"`
   - Rename `spread_line` -> `nflverse_spread_line`, `total_line` -> `nflverse_total_line` (per D-17)
   - Log orphan count (rows where game_id is NaN) per D-13
   - Return merged DataFrame

9. **`validate_cross_correlation(df: pd.DataFrame) -> bool`** -- per D-19:
   - Drop rows with NaN in closing_spread or nflverse_spread_line
   - Compute `pearsonr(closing_spread, nflverse_spread_line)` -- expect r > 0.95
   - Compute `(abs(closing_spread - nflverse_spread_line) <= 1.0).mean()` -- expect > 0.95
   - Raise ValueError if either threshold fails
   - Print results

10. **`validate_sign_convention(df: pd.DataFrame) -> bool`** -- per D-21:
    - Filter rows where `nflverse_spread_line > 7` (clear home favorites in nflverse convention)
    - Assert `opening_spread > 0` for all such rows (after negation, home favorites should be positive)
    - Raise ValueError if any sign flip found
    - Return True

11. **`validate_odds_schema(df: pd.DataFrame) -> bool`** -- per D-08:
    - Required columns: `["game_id", "season", "week", "game_type", "home_team", "away_team", "opening_spread", "closing_spread", "opening_total", "closing_total", "home_moneyline", "away_moneyline", "nflverse_spread_line", "nflverse_total_line"]`

    - Return True

12. **`validate_row_counts(df: pd.DataFrame, season: int) -> None`** -- per D-20:
    - Expected: 256 regular season games pre-2021, 272 post-2021, plus playoffs

13. **`write_parquet(df: pd.DataFrame, season: int, dry_run: bool = False) -> str`** -- per project pattern:
    - Select final columns: the 14 required columns from D-15/D-16/D-17
    - Output path: `data/bronze/odds/season={season}/odds_{YYYYMMDD}_{HHMMSS}.parquet`
