# Requirements: NFL Data Engineering Platform

**Defined:** 2026-03-27
**Core Value:** A rich, well-modeled NFL data lake that serves as the foundation for both fantasy football decision-making and game prediction models.

## v2.1 Requirements

Requirements for v2.1 Market Data milestone. Each maps to roadmap phases.

### Odds Ingestion

- [x] **ODDS-01**: Download and parse SBRO XLSX archives into Parquet with opening/closing spreads and totals (2016-2021) (implemented via FinnedAI JSON — see v2.1-MILESTONE-AUDIT.md)
- [x] **ODDS-02**: Map SBRO team names to nflverse game_id with validated team name mapping dictionary (implemented via FinnedAI JSON — see v2.1-MILESTONE-AUDIT.md)
- [x] **ODDS-03**: Register 'odds' as a Bronze data type with schema validation in the ingestion registry

### Line Movement

- [x] **LINE-01**: Compute spread movement (closing - opening) and total movement per game with nflverse sign convention
- [x] **LINE-02**: Categorize movement into direction buckets (large >2pts, medium 1-2, small <1, none)
- [x] **LINE-03**: Detect steam moves (sharp money indicators) where data supports it; NaN where timestamps unavailable
- [x] **LINE-04**: Add line movement features as candidates to feature selection with ablation on sealed holdout

### CLV Tracking

- [x] **CLV-01**: Compute point-based CLV (model_spread - closing_spread) per game in backtest output
- [x] **CLV-02**: Report average CLV broken out by confidence tier (high/medium/low) in backtest summary
- [x] **CLV-03**: Track per-season CLV averages to measure model quality trends over time

## Out of Scope

| Feature | Reason |
|---------|--------|
| Real-time odds API integration | Free tiers too limited for historical backfill; SBRO covers training window |
| No-vig line calculation | Adds complexity; defer to v2.2 betting framework |
| Live line movement tracking | Requires streaming API; batch historical is sufficient for prediction |
| Neural networks / deep learning | Gradient boosting dominates tabular sports prediction at this data scale |
| Closing line as prediction feature | Leakage -- closing line is NOT pre-game knowable |
